# Sales & Transaction Data Analysis 📊

## Overview
This project analyzes retail sales and transaction data to identify trends, detect discrepancies, and improve operational accuracy.

## Tools Used
- Excel
- SQL
- Power BI

## Key Features
- Data cleaning and validation
- Sales trend analysis
- Dashboard visualization
- Insights for decision-making

## Dataset
Sample retail sales dataset sourced from Kaggle for analysis practice.

## Results
- Identified revenue patterns
- Detected inconsistencies in transactions
- Improved reporting accuracy

## How to Run
1. Download dataset
2. Open Power BI / Excel file
3. View dashboard
